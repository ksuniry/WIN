비밀번호 관리 안내

마이머플러 비밀번호를 안전하게 관리하는 방법

마이머플러를 안전하게 이용하기 위해서는 추측하기 어려운 비밀번호를 만들고, 어떤 경로를 통해서든 노출되지 않도록 하여 다른 사람이 비밀번호를 알 수 없도록 관리해야 합니다. 자신만의 규칙을 이용하여 자주 변경하는 것도 도움이 되며, 분실/도난/복제 가능성이 있는 인증 수단은 별도의 안전한 곳에 보관하거나 항상 몸에 지니고 다니는 등 관리가 필요합니다.

1. 다른 금융기관, 사이트 등에서 사용한 적이 없고 이전에 사용한 적이 없는 비밀번호가 안전합니다. 나만의 규칙을 이용하여 자주 비밀번호를 변경하면 더욱 안전합니다.

2. 계좌번호, 주소, 전화번호, 차량번호 또는 연속된 숫자 등 다른 사람이 쉽게 알아낼 수 있는 비밀번호 사용은 위험합니다.

3. 비밀번호를 통장, 수첩 등에 적어두거나 휴대기기/PC/이메일 등에 저장하지 마세요.

4. 무엇보다 머플러만의 비밀번호를 사용하는 것이 가장 중요합니다.

5. 계좌번호, 비밀번호, PIN번호 등의 개인 정보가 다른 사람에게 노출되지 않도록 관리해 주세요.

6. 웰스가이드 직원을 포함하여 누구에게도 비밀번호, PIN 번호등을 알려주지 마세요.

7. 사용 후에는 반드시 로그아웃 해주세요.

! 아래와 같은 비밀번호는 사용하지 마세요.

1. 빈 칸/공백
2. 연속되거나 동일한 숫자를 많이 포함하는 비밀번호
3. 생일 등의 개인 정보로 만 이뤄진 비밀번호
4. 흔히 많이 사용하는 것으로 알려진 쉬운 비밀번호
5. 도용 피해가 발생하여 타인에게 알려졌다고 의심되거나 확인된 비밀번호
